HTML Text Formatting Guide

Basic Text Formatting

HTML provides various elements for formatting text to enhance readability and emphasize content. These formatting options help create visually appealing and well-structured documents.

Bold Text

There are two ways to make text bold in HTML: the <strong> tag and the <b> tag. The <strong> tag is semantic and indicates strong importance, while <b> is purely stylistic.

Code: <strong>Important text</strong>

Result: Important text

Italic Text

For italic text, you can use either the <em> tag for emphasis or the <i> tag for stylistic purposes. The <em> tag is preferred when the text needs to be emphasized semantically.

Code: <em>Emphasized text</em>

Result: Emphasized text

Underlined Text

The <u> tag is used to underline text. However, it's important to note that underlining text should be used sparingly as it can be confused with hyperlinks.

Code: <u>Underlined text</u>

Result: Underlined text

Strikethrough Text

The <del> or <s> tags can be used to create strikethrough text, indicating that content has been deleted or is no longer valid.

Code: <del>Deleted text</del>

Result: Deleted text

Subscript and Superscript

Use <sub> for subscript and <sup> for superscript text. These are commonly used in mathematical equations or chemical formulas.

Code: H<sub>2</sub>O and X<sup>2</sup>

Result: H2O and X2

Highlighted Text

The <mark> tag is used to highlight text, similar to using a highlighter pen on paper.

Code: <mark>Highlighted text</mark>

Result: Highlighted text

Text Alignment

While text alignment is typically handled with CSS, HTML provides basic alignment capabilities through the style attribute or CSS classes. Text can be aligned left, right, center, or justified.

Code: <p style="text-align: center">Centered text</p>

Result: Centered text

Best Practices

When formatting text in HTML, remember these key points:

  • Use semantic elements when possible (<strong> over <b>)
  • Avoid overusing text formatting
  • Consider accessibility when choosing formatting options
  • Use CSS for styling when possible
  • Maintain consistency throughout your document
