HTML Links & Images Guide

HTML Links (Hyperlinks)

Links are one of the most fundamental elements of HTML, allowing users to navigate between pages and websites. Links are created using the <a> (anchor) tag.

<a href="https://www.example.com">Visit Example.com</a>

Types of Links

Type Example
Absolute URLs <a href="https://www.example.com">External Link</a>
Relative URLs <a href="about.html">Internal Link</a>
Email Links <a href="mailto:example@email.com">Send Email</a>
Phone Links <a href="tel:+1234567890">Call Us</a>

Link Attributes

Important attributes for links include:

  • href - Specifies the destination URL
  • target - Specifies where to open the link (_blank, _self, etc.)
  • title - Provides additional information about the link
  • rel - Specifies the relationship between current and linked document

HTML Images

Images are added to HTML pages using the <img> tag. This is a self-closing tag that requires a source (src) attribute to specify the image location.

<img src="image.jpg" alt="Description of image" width="300" height="200">

Image Attributes

  • src - Required. Specifies the path to the image
  • alt - Required. Provides alternative text for accessibility
  • width - Specifies the width of the image
  • height - Specifies the height of the image
  • loading - Specifies how the browser should load the image

Image Best Practices

Important Tips:

  • Always include alt text for accessibility
  • Optimize images for web use
  • Use appropriate image formats (JPG, PNG, WebP)
  • Specify image dimensions to prevent layout shifts
  • Consider responsive images using srcset

Responsive Images

Make images responsive using CSS:

img { max-width: 100%; height: auto; }

Image Formats

Format Best Use Case
JPEG/JPG Photographs and complex images with many colors
PNG Images requiring transparency or screenshots
WebP Modern format offering better compression
SVG Logos, icons, and scalable graphics

Combining Links and Images

You can wrap images in anchor tags to create clickable images:

<a href="https://example.com"> <img src="image.jpg" alt="Click me" width="200"> </a>
