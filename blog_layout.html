Blog Layout Guide

Basic Blog Structure

A well-structured blog should include these essential elements:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Blog Title</title> </head> <body> <header> <nav> <!-- Navigation Menu --> </nav> </header> <main class="blog-content"> <article class="post"> <!-- Blog Post Content --> </article> <aside class="sidebar"> <!-- Sidebar Content --> </aside> </main> <footer> <!-- Footer Content --> </footer> </body> </html>

Blog Post Structure

<article class="post"> <header class="post-header"> <h2 class="post-title">Post Title</h2> <div class="post-meta"> <time datetime="2024-01-01">January 1, 2024</time> <span class="author">By Author Name</span> </div> </header> <div class="post-content"> <img src="post-image.jpg" alt="Post Image"> <p>Post content goes here...</p> </div> <footer class="post-footer"> <div class="tags"> <a href="#">#tag1</a> <a href="#">#tag2</a> </div> </footer> </article>

Responsive Design

@media (max-width: 768px) { .blog-content { grid-template-columns: 1fr; } .sidebar { margin-top: 20px; } .blog-grid { grid-template-columns: 1fr; } }

Pagination

<nav class="pagination"> <ul> <li><a href="#" class="prev">Previous</a></li> <li><a href="#">1</a></li> <li><a href="#">2</a></li> <li><a href="#">3</a></li> <li><a href="#" class="next">Next</a></li> </ul> </nav>

Best Practices

  • Use semantic HTML elements
  • Implement responsive design
  • Optimize images
  • Include proper meta tags
  • Add social sharing buttons
  • Implement search functionality
  • Use categories and tags
  • Add related posts section

SEO Considerations

  • Use descriptive URLs
  • Implement proper heading hierarchy
  • Add meta descriptions
  • Use alt text for images
  • Create XML sitemap
  • Optimize loading speed

Performance Tips

  • Lazy load images
  • Minimize HTTP requests
  • Use appropriate image formats
  • Implement caching
  • Minify CSS and JavaScript
  • Use CDN for assets
