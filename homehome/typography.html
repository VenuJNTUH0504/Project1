CSS Typography

Typography in CSS allows you to control the appearance of text on your web pages. You can set font families, sizes, weights, styles, line heights, and more to create visually appealing and readable content.

Common Typography Properties

1. Font Family

The font-family property specifies the typeface for text.

This text uses Courier New font.
font-family: 'Courier New', Courier, monospace;

2. Font Size

The font-size property sets the size of the text.

This is large text.
font-size: 2rem;

3. Font Weight

The font-weight property sets the thickness of the text.

This is bold text.
font-weight: bold;

4. Font Style

The font-style property is used for italic or oblique text.

This is italic text.
font-style: italic;

5. Line Height

The line-height property controls the vertical spacing between lines of text.

This text has increased line height.
Notice the extra space between lines.
line-height: 2.5;

6. Text Align

The text-align property sets the horizontal alignment of text.

This text is centered.
text-align: center;

Typography Properties Comparison

  • font-family: Sets the typeface
  • font-size: Sets the size of the text
  • font-weight: Sets the thickness (boldness)
  • font-style: Sets italic or normal style
  • line-height: Sets space between lines
  • text-align: Sets horizontal alignment
