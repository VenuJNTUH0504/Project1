CSS Units

CSS uses different units to express length, size, and spacing. Understanding these units helps you create responsive and scalable layouts.

Common CSS Units

1. px (Pixels)

16px
Absolute unit. 1px equals one device pixel. Commonly used for precise control.
font-size: 16px;

2. em

2em
Relative to the font-size of the element. 2em is twice the current font size.
font-size: 2em;

3. rem

2rem
Relative to the root element's font-size. Consistent across the document.
font-size: 2rem;

4. % (Percent)

60%
Relative to the parent element. Useful for responsive layouts.
width: 60%;

5. vw / vh

20vw
Viewport width (vw) and height (vh). 1vw is 1% of viewport width.
width: 20vw;

6. fr (Fractional Unit)

1fr
Used in CSS Grid to distribute space. 1fr means one fraction of available space.
grid-template-columns: 1fr 2fr;

Units Comparison

  • px: Absolute, fixed size
  • em: Relative to element's font size
  • rem: Relative to root font size
  • %: Relative to parent
  • vw/vh: Relative to viewport
  • fr: Fraction of available space (Grid)
