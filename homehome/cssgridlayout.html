CSS Grid Layout

CSS Grid Layout is a powerful two-dimensional layout system for the web. It allows you to create complex layouts easily and precisely using rows and columns.

Basic Grid Example

To create a grid, set display: grid; on a container and define columns and rows.

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

Grid Areas

Grid areas allow you to name sections of your layout and place items precisely.

Header
Sidebar
Content
Footer
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar content"
    "footer footer";
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

Common Grid Properties

  • display: grid; — Enables grid layout on a container
  • grid-template-columns — Defines the number and size of columns
  • grid-template-rows — Defines the number and size of rows
  • gap — Sets spacing between grid items
  • grid-area — Assigns a name to a grid item for placement
  • justify-items / align-items — Aligns items within their grid cells
  • grid-template-areas — Defines named areas for layout
