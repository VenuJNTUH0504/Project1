CSS Pseudo-classes & Elements

Pseudo-classes and pseudo-elements allow you to style elements based on their state or position, or to style specific parts of elements.

Common Pseudo-classes

1. :hover

Applies a style when the user hovers over an element.

Hover over this link
a:hover { color: #f39c12; }

2. :focus

Applies a style when an element (like an input) has focus.


input:focus { border: 2px solid #4a90e2; }

3. :first-child

Targets the first child element within its parent.

  • First item
  • Second item
  • Third item
li:first-child { font-weight: bold; }

Common Pseudo-elements

1. ::before and ::after

Used to insert content before or after an element's actual content.

This paragraph uses ::before and ::after.

p::before { content: "★ "; }
p::after { content: " ✔"; }

2. ::first-line

Styles the first line of a block of text.

This is a paragraph. The first line will be styled differently using ::first-line pseudo-element.

p::first-line { font-weight: bold; color: #4a90e2; }

Pseudo-classes & Elements Comparison

  • :hover: Styles on mouse hover
  • :focus: Styles when element is focused
  • :first-child: Styles the first child of a parent
  • ::before: Inserts content before element
  • ::after: Inserts content after element
  • ::first-line: Styles the first line of text
