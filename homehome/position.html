CSS Position Property

The position property in CSS specifies how an element is positioned in a document. It is a key property for controlling layout and stacking of elements.

Common Position Values

1. Static

This is the default value. Elements are positioned according to the normal flow of the document.

Static Position
position: static;

2. Relative

The element is positioned relative to its normal position. Use top, right, bottom, and left to move it.

Relative Position (left: 30px; top: 10px;)
position: relative; left: 30px; top: 10px;

3. Absolute

The element is positioned absolutely to its nearest positioned ancestor (not static). If none, it uses the document body.

Absolute Position (left: 30px; top: 10px;)
position: absolute; left: 30px; top: 10px;

4. Fixed

The element is positioned relative to the browser window. It stays in the same place even when the page is scrolled.

Fixed Position (left: 30px; top: 120px;)
position: fixed; left: 30px; top: 120px;

5. Sticky

The element toggles between relative and fixed, depending on the scroll position.

Sticky Position (top: 10px;)
position: sticky; top: 10px;

Position Properties Comparison

  • Static: Default, follows normal flow
  • Relative: Offset from normal position
  • Absolute: Positioned to nearest ancestor
  • Fixed: Positioned relative to viewport
  • Sticky: Scrolls until a threshold, then sticks
