CSS Transitions & Transforms

CSS Transitions and Transforms allow you to create smooth, interactive visual effects and animations on your web elements.

Key Concepts

1. CSS Transitions

Transitions enable you to change property values smoothly (over a given duration) instead of instantly.

Hover me!
transition: background 0.5s, color 0.5s;

2. CSS Transforms

Transforms let you rotate, scale, skew, or move elements in 2D or 3D space.

Hover me!
transform: scale(1.2) rotate(10deg);

3. Combining Transitions & Transforms

You can combine transitions and transforms for more dynamic effects.

Hover me!
transition: transform 0.5s, background 0.5s;
transform: translateY(-10px) scale(1.1);

Common Properties

  • transition-property: The CSS property you want to animate (e.g., background, transform).
  • transition-duration: How long the transition takes (e.g., 0.5s).
  • transition-timing-function: The speed curve (e.g., ease, linear).
  • transform: Functions like scale(), rotate(), translate(), skew().
