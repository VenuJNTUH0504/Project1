CSS Selectors

CSS selectors are patterns used to select and style HTML elements. They are fundamental to targeting elements in your stylesheets.

Common CSS Selectors

1. Element Selector

Selects all elements of a given type.

This is a <span> element span { color: blue; }

2. Class Selector

Selects all elements with a specific class attribute.

Class Example .my-class { color: orange; }

3. ID Selector

Selects a single element with a specific id attribute.

ID Example #my-id { color: green; }

4. Descendant Selector

Selects elements that are descendants of another element.

Descendant Example

p span { color: purple; }

5. Attribute Selector

Selects elements with a specific attribute or attribute value.

Attribute Example span[data-role="admin"] { color: red; }

Selector Types Comparison

  • Element: Targets all elements of a type (e.g., p, div)
  • Class: Targets elements with a class (e.g., .my-class)
  • ID: Targets a unique element by id (e.g., #my-id)
  • Descendant: Targets elements inside other elements (e.g., ul li)
  • Attribute: Targets elements with attributes (e.g., input[type="text"])
