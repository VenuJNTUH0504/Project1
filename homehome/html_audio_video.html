HTML Audio & Video Guide

HTML Audio Element

The HTML <audio> element is used to embed sound content in documents. It may contain one or more audio sources, represented using the src attribute or the <source> element.

<audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> Your browser does not support the audio element. </audio>

Audio Attributes

Attribute Description
controls Shows the audio player controls
autoplay Starts playing audio automatically
loop Repeats the audio when finished
muted Sets the audio output to muted
preload Specifies if audio should be loaded when page loads

HTML Video Element

The HTML <video> element is used to embed video content. Like audio, it can contain multiple sources and provides various controls for playback.

<video width="100%" controls> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> Your browser does not support the video element. </video>

Video Attributes

Attribute Description
width Sets the width of the video player
height Sets the height of the video player
poster Specifies an image to show before the video plays
controls Shows the video player controls
autoplay Starts playing video automatically

Supported Media Formats

Audio Formats:

  • MP3 (.mp3)
  • WAV (.wav)
  • OGG (.ogg)

Video Formats:

  • MP4 (.mp4)
  • WebM (.webm)
  • OGG (.ogv)

Best Practices

  • Always provide multiple source formats for better browser compatibility
  • Include fallback content for browsers that don't support the media
  • Consider bandwidth and loading times when embedding media
  • Use appropriate video dimensions for your website layout
  • Implement responsive design for media elements
  • Consider accessibility features like captions and transcripts

Responsive Media

Make media elements responsive using CSS:

video, audio { max-width: 100%; height: auto; }

Embedding External Media

You can also embed videos from external sources like YouTube using iframes:

<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen> </iframe>
