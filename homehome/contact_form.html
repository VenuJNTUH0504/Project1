Basic Contact Form Structure

<form action="/submit" method="POST"> <div class="form-group"> <label for="name">Name:</label> <input type="text" id="name" name="name" required> </div> <div class="form-group"> <label for="email">Email:</label> <input type="email" id="email" name="email" required> </div> <div class="form-group"> <label for="message">Message:</label> <textarea id="message" name="message" required></textarea> </div> <button type="submit">Send Message</button> </form>

Form Validation

<input type="text" pattern="[A-Za-z ]+" title="Letters only"> <input type="email" required> <input type="tel" pattern="[0-9]{10}" title="10 digits">

Form Styling

.form-group { margin-bottom: 20px; } input, textarea { width: 100%; padding: 10px; border: 1px solid #ddd; border-radius: 4px; } button { background: #7F7FD5; color: white; padding: 12px 24px; border: none; border-radius: 4px; }

Best Practices

  • Use appropriate input types
  • Include proper form validation
  • Add clear error messages
  • Make forms mobile-responsive
  • Use HTTPS for secure data transmission
  • Include proper form labels
  • Implement CSRF protection
  • Add success/error feedback

Accessibility Features

  • Use proper label associations
  • Include ARIA attributes
  • Provide clear error messages
  • Ensure keyboard navigation
  • Use sufficient color contrast
  • Add focus indicators

Security Considerations

  • Implement server-side validation
  • Use CSRF tokens
  • Sanitize input data
  • Use HTTPS
  • Implement rate limiting
  • Add CAPTCHA for spam prevention
