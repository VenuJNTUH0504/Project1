CSS Display Types

The display property specifies how an element should be displayed. It's one of the most important CSS properties for controlling layout.

Common Display Values

1. Block

Block elements start on a new line and take up the full width available.

Block Element 1
Block Element 2
display: block;

2. Inline

Inline elements only take up as much width as necessary and don't force new lines.

Inline Element 1 Inline Element 2
display: inline;

3. Flex

Flexbox is a one-dimensional layout method for arranging items in rows or columns.

Flex Item 1
Flex Item 2
Flex Item 3
display: flex;

4. Grid

CSS Grid is a two-dimensional layout system for the web.

Grid Item 1
Grid Item 2
Grid Item 3
Grid Item 4
Grid Item 5
Grid Item 6
display: grid;

5. None

Elements with display: none are completely removed from the layout.

display: none;

Display Properties Comparison

  • Block: Full width, starts on new line
  • Inline: Only takes necessary width, no line breaks
  • Inline-block: Like inline but can have width and height
  • Flex: Flexible box layout
  • Grid: Two-dimensional grid layout
  • None: Element is not displayed
