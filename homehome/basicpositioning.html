CSS Positioning Basics

The position property in CSS determines how an element is positioned in the document. It is essential for creating layouts and controlling the placement of elements.

Common Position Values

1. Static

This is the default position. Elements are positioned according to the normal flow of the document.

Static
position: static;

2. Relative

The element is positioned relative to its normal position. Use top, right, bottom, and left to move it.

Relative
position: relative; left: 20px; top: 20px;

3. Absolute

The element is positioned absolutely to its nearest positioned ancestor (not static). It is removed from the normal flow.

Absolute
position: absolute; left: 20px; top: 20px;

4. Fixed

The element is positioned relative to the browser window. It stays in the same place even when the page is scrolled.

Fixed
position: fixed; left: 20px; top: 20px;

5. Sticky

The element toggles between relative and fixed, depending on the scroll position.

Sticky
position: sticky; top: 10px;

Position Properties Comparison

  • Static: Default, follows normal flow
  • Relative: Offset from normal position
  • Absolute: Positioned to nearest ancestor, removed from flow
  • Fixed: Positioned relative to viewport, stays on scroll
  • Sticky: Scrolls until a threshold, then sticks
