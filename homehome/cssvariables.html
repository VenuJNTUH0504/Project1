CSS Variables

CSS Variables, also known as custom properties, allow you to store values in one place and reuse them throughout your stylesheet. They make your CSS more maintainable and flexible.

How to Declare and Use CSS Variables

1. Declaring Variables

Variables are declared inside a selector using the --variable-name syntax, commonly in the :root selector for global scope.

:root {
  --main-bg: #4a90e2;
  --accent: #f39c12;
  --padding: 16px;
}

2. Using Variables

Use the var() function to access the value of a variable.

body {
  background: var(--main-bg);
  color: #fff;
  padding: var(--padding);
}

3. Variable Scope

Variables can be scoped to the :root (global) or to specific selectors (local).

:root {
  --main-color: #4a90e2;
}
.button {
  --main-color: #f39c12;
  color: var(--main-color);
}

4. Fallback Values

You can provide a fallback value if the variable is not defined.

color: var(--text-color, #222);

CSS Variables vs. Preprocessor Variables

  • CSS Variables: Native to CSS, dynamic, can be changed at runtime with JavaScript
  • Preprocessor Variables (e.g., SASS/LESS): Static, replaced at compile time, not accessible in the browser

Benefits of CSS Variables

  • Easy to maintain and update values
  • Enable theming and dynamic changes
  • Reduce code duplication
  • Work with JavaScript for live updates
