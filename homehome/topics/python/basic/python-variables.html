Python Variables and Data Types

Understanding variables and data types in Python

Variables in Python

Variables are containers for storing data values. In Python, variables are created when you assign a value to them:

# Variable assignment
name = "John"
age = 25
height = 1.75
is_student = True

Python is dynamically typed, which means you don't need to declare the type of a variable when you create it.

Basic Data Types

Data Type Description Example
int Integer numbers 42
float Floating-point numbers 3.14
str Text strings "Hello"
bool Boolean values True or False

Type Conversion

Python provides built-in functions to convert between different data types:

# Converting between types
number = 42
float_number = float(number)  # Convert to float
string_number = str(number)   # Convert to string
boolean_value = bool(number)  # Convert to boolean

Variable Naming Rules

  • Variable names can only contain letters, numbers, and underscores
  • Variable names cannot start with a number
  • Variable names are case-sensitive
  • Variable names should be descriptive and meaningful
  • Use snake_case for variable names (e.g., user_name)

Avoid using Python keywords as variable names (e.g., if, for, while).

Multiple Assignment

Python allows you to assign values to multiple variables in one line:

# Multiple assignment
x, y, z = 1, 2, 3

# Swapping variables
a = 5
b = 10
a, b = b, a  # Swap values

Constants

While Python doesn't have built-in constants, it's a convention to use uppercase letters for variables that should be treated as constants:

# Constants (convention)
PI = 3.14159
GRAVITY = 9.81
MAX_CONNECTIONS = 100
