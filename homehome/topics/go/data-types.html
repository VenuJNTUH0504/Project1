Go Data Types & Variables

Understanding Go's type system and variable declarations

Basic Types

Go has several built-in basic types:

Type Description Example
bool Boolean value (true or false) var b bool = true
string Text string var s string = "Hello"
int Signed integer (32 or 64 bits) var i int = 42
float64 64-bit floating-point number var f float64 = 3.14
complex128 Complex number with float64 real and imaginary parts var c complex128 = 1 + 2i

Variable Declaration

Go provides several ways to declare variables:

// Explicit type declaration
var name string = "John"

// Type inference
var age = 30

// Short declaration (inside functions)
height := 180

// Multiple variables
var (
    firstName = "John"
    lastName  = "Doe"
    age       = 30
)

// Constants
const pi = 3.14159

Zero Values

Variables declared without an explicit initial value are given their zero value:

Type Zero Value
bool false
string ""
int 0
float64 0.0
complex128 0 + 0i

Type Conversion

Go requires explicit type conversion:

var i int = 42
var f float64 = float64(i)
var u uint = uint(f)

// Type conversion between numeric types
var x int = 10
var y float64 = float64(x)

// String conversion
var s string = string(65) // "A"
var num string = string(48) // "0"

Best Practices

  • Use meaningful variable names
  • Prefer short declarations inside functions
  • Use constants for values that don't change
  • Be explicit with type conversions
  • Use appropriate numeric types for your needs
  • Initialize variables when possible
  • Use the zero value when appropriate
