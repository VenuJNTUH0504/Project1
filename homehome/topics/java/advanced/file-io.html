File I/O in Java

Understanding file operations and I/O streams in Java

Introduction to File I/O

Java provides a rich set of classes for handling file operations and I/O streams.

Key I/O concepts:

  • Streams (byte and character streams)
  • Readers and Writers
  • Buffered I/O
  • File operations
  • NIO (New I/O) package

Basic File Operations

Java provides the File class for basic file operations.

Example: File Operations

import java.io.File;

public class FileOperations {
    public static void main(String[] args) {
        // Create File object
        File file = new File("example.txt");
        
        // Check if file exists
        if (file.exists()) {
            System.out.println("File exists");
            
            // Get file information
            System.out.println("File name: " + file.getName());
            System.out.println("Absolute path: " + file.getAbsolutePath());
            System.out.println("File size: " + file.length() + " bytes");
            
            // Delete file
            if (file.delete()) {
                System.out.println("File deleted");
            }
        } else {
            // Create new file
            try {
                if (file.createNewFile()) {
                    System.out.println("File created");
                }
            } catch (IOException e) {
                e.printStackTrace();
            }
        }
    }
}

Byte Streams

Byte streams handle raw binary data.

Example: FileInputStream and FileOutputStream

import java.io.*;

public class ByteStreamExample {
    public static void main(String[] args) {
        // Writing to a file
        try (FileOutputStream fos = new FileOutputStream("data.bin")) {
            byte[] data = {65, 66, 67, 68, 69}; // ABCDE
            fos.write(data);
        } catch (IOException e) {
            e.printStackTrace();
        }

        // Reading from a file
        try (FileInputStream fis = new FileInputStream("data.bin")) {
            int byteData;
            while ((byteData = fis.read()) != -1) {
                System.out.print((char) byteData);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Character Streams

Character streams handle text data.

Example: FileReader and FileWriter

import java.io.*;

public class CharacterStreamExample {
    public static void main(String[] args) {
        // Writing to a file
        try (FileWriter writer = new FileWriter("text.txt")) {
            writer.write("Hello, World!\n");
            writer.write("This is a test file.");
        } catch (IOException e) {
            e.printStackTrace();
        }

        // Reading from a file
        try (FileReader reader = new FileReader("text.txt")) {
            int charData;
            while ((charData = reader.read()) != -1) {
                System.out.print((char) charData);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Buffered Streams

Buffered streams improve I/O performance by reducing the number of I/O operations.

Example: Buffered I/O

import java.io.*;

public class BufferedStreamExample {
    public static void main(String[] args) {
        // Writing with buffered stream
        try (BufferedWriter writer = new BufferedWriter(new FileWriter("buffered.txt"))) {
            writer.write("Line 1\n");
            writer.write("Line 2\n");
            writer.write("Line 3");
        } catch (IOException e) {
            e.printStackTrace();
        }

        // Reading with buffered stream
        try (BufferedReader reader = new BufferedReader(new FileReader("buffered.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

NIO (New I/O)

NIO provides a more efficient way to handle I/O operations.

Example: NIO File Operations

import java.nio.file.*;
import java.nio.charset.StandardCharsets;
import java.io.IOException;

public class NIOExample {
    public static void main(String[] args) {
        // Writing to a file
        try {
            Path path = Paths.get("nio.txt");
            String content = "Hello, NIO!";
            Files.write(path, content.getBytes(StandardCharsets.UTF_8));
            
            // Reading from a file
            byte[] bytes = Files.readAllBytes(path);
            String readContent = new String(bytes, StandardCharsets.UTF_8);
            System.out.println(readContent);
            
            // Copy file
            Path copyPath = Paths.get("nio_copy.txt");
            Files.copy(path, copyPath, StandardCopyOption.REPLACE_EXISTING);
            
            // Delete file
            Files.deleteIfExists(copyPath);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

File I/O Best Practices

Best practices for file I/O:

  • Always close resources using try-with-resources
  • Use buffered streams for better performance
  • Handle exceptions appropriately
  • Use NIO for modern file operations
  • Check file existence before operations
  • Use appropriate character encoding
  • Consider file locking for concurrent access
