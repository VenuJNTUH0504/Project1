Arrays & Strings in Java

Understanding arrays and string manipulation in Java

Basic Arrays

Creating and working with one-dimensional arrays.

public class BasicArrays {
    public static void main(String[] args) {
        // Array declaration and initialization
        int[] numbers = new int[5];  // Array of 5 integers
        String[] names = {"Alice", "Bob", "Charlie"};  // Array with initial values
        
        // Accessing array elements
        numbers[0] = 10;
        numbers[1] = 20;
        
        // Array length
        System.out.println("Array length: " + numbers.length);
        
        // Iterating through array
        for (int i = 0; i < numbers.length; i++) {
            System.out.println("Element " + i + ": " + numbers[i]);
        }
        
        // Enhanced for loop
        for (String name : names) {
            System.out.println("Name: " + name);
        }
    }
}

Note: Array indices start at 0 and go up to length-1.

Multidimensional Arrays

Working with two-dimensional and three-dimensional arrays.

public class MultiDimensionalArrays {
    public static void main(String[] args) {
        // 2D array
        int[][] matrix = new int[3][3];
        
        // Initialize 2D array
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                matrix[i][j] = i + j;
            }
        }
        
        // Access elements
        System.out.println("Element at [1][2]: " + matrix[1][2]);
        
        // 3D array
        int[][][] cube = new int[2][2][2];
        
        // Initialize 3D array
        for (int i = 0; i < cube.length; i++) {
            for (int j = 0; j < cube[i].length; j++) {
                for (int k = 0; k < cube[i][j].length; k++) {
                    cube[i][j][k] = i + j + k;
                }
            }
        }
    }
}

Warning: Be careful with array bounds to avoid ArrayIndexOutOfBoundsException.

String Basics

Basic string operations and manipulation.

public class StringBasics {
    public static void main(String[] args) {
        // String creation
        String str1 = "Hello";
        String str2 = new String("World");
        
        // String concatenation
        String combined = str1 + " " + str2;
        System.out.println(combined);
        
        // String length
        System.out.println("Length: " + combined.length());
        
        // Character access
        char firstChar = combined.charAt(0);
        System.out.println("First character: " + firstChar);
        
        // Substring
        String sub = combined.substring(0, 5);
        System.out.println("Substring: " + sub);
        
        // String comparison
        boolean isEqual = str1.equals("Hello");
        System.out.println("Strings equal: " + isEqual);
        
        // Case conversion
        String upper = combined.toUpperCase();
        String lower = combined.toLowerCase();
        System.out.println("Upper: " + upper);
        System.out.println("Lower: " + lower);
    }
}

Note: Strings are immutable in Java. Operations that modify strings create new string objects.

StringBuilder and StringBuffer

Using mutable string classes for efficient string manipulation.

public class StringBuilders {
    public static void main(String[] args) {
        // StringBuilder (not thread-safe but faster)
        StringBuilder sb = new StringBuilder();
        sb.append("Hello");
        sb.append(" ");
        sb.append("World");
        System.out.println(sb.toString());
        
        // StringBuffer (thread-safe)
        StringBuffer sbf = new StringBuffer();
        sbf.append("Thread");
        sbf.append(" ");
        sbf.append("Safe");
        System.out.println(sbf.toString());
        
        // Common operations
        sb.insert(5, ", ");
        System.out.println("After insert: " + sb);
        
        sb.delete(5, 7);
        System.out.println("After delete: " + sb);
        
        sb.reverse();
        System.out.println("Reversed: " + sb);
    }
}

Best Practice: Use StringBuilder for single-threaded applications and StringBuffer for multi-threaded applications.

String Formatting

Formatting strings using String.format() and printf().

public class StringFormatting {
    public static void main(String[] args) {
        // Basic formatting
        String formatted = String.format("Hello, %s! You are %d years old.", "Alice", 25);
        System.out.println(formatted);
        
        // Number formatting
        double pi = Math.PI;
        System.out.printf("Pi to 2 decimal places: %.2f%n", pi);
        System.out.printf("Pi in scientific notation: %e%n", pi);
        
        // Date formatting
        java.util.Date now = new java.util.Date();
        System.out.printf("Current date: %tF%n", now);
        System.out.printf("Current time: %tT%n", now);
        
        // Padding and alignment
        System.out.printf("|%10s|%n", "Hello");  // Right-aligned
        System.out.printf("|%-10s|%n", "Hello"); // Left-aligned
        System.out.printf("|%010d|%n", 42);      // Zero-padded
    }
}

Note: Format specifiers start with % and include flags, width, precision, and conversion characters.

String Tokenization

Splitting strings into tokens using StringTokenizer and split().

import java.util.StringTokenizer;

public class StringTokenization {
    public static void main(String[] args) {
        String text = "Hello,World,Java";
        
        // Using split()
        String[] parts = text.split(",");
        for (String part : parts) {
            System.out.println("Split: " + part);
        }
        
        // Using StringTokenizer
        StringTokenizer tokenizer = new StringTokenizer(text, ",");
        while (tokenizer.hasMoreTokens()) {
            System.out.println("Token: " + tokenizer.nextToken());
        }
        
        // Complex splitting
        String complexText = "Hello, World; Java|Programming";
        String[] complexParts = complexText.split("[,;|]");
        for (String part : complexParts) {
            System.out.println("Complex split: " + part.trim());
        }
    }
}

Warning: Regular expressions in split() can be complex. Test thoroughly.

String Searching and Manipulation

Advanced string operations including searching and pattern matching.

public class StringSearching {
    public static void main(String[] args) {
        String text = "Hello World, Hello Java";
        
        // Finding index
        int index = text.indexOf("World");
        System.out.println("Index of 'World': " + index);
        
        // Last occurrence
        int lastIndex = text.lastIndexOf("Hello");
        System.out.println("Last index of 'Hello': " + lastIndex);
        
        // Contains check
        boolean contains = text.contains("Java");
        System.out.println("Contains 'Java': " + contains);
        
        // Starts with and ends with
        boolean starts = text.startsWith("Hello");
        boolean ends = text.endsWith("Java");
        System.out.println("Starts with 'Hello': " + starts);
        System.out.println("Ends with 'Java': " + ends);
        
        // Replace
        String replaced = text.replace("Hello", "Hi");
        System.out.println("Replaced: " + replaced);
        
        // Regular expressions
        String regex = "\\b\\w{5}\\b";  // Match 5-letter words
        String[] matches = text.split(regex);
        for (String match : matches) {
            System.out.println("Regex match: " + match);
        }
    }
}

Best Practice: Use regular expressions for complex pattern matching and string manipulation.
