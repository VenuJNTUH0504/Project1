CSS Grid Layout

.gallery { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; padding: 20px; }

Image Hover Effects

.gallery-item { transition: transform 0.3s ease; } .gallery-item:hover { transform: scale(1.05); }

Lightbox Implementation

<div class="lightbox" id="lightbox"> <img src="" alt="Lightbox Image"> </div> <script> // Lightbox functionality const lightbox = document.getElementById('lightbox'); const images = document.querySelectorAll('.gallery-item img'); images.forEach(image => { image.addEventListener('click', () => { lightbox.style.display = 'flex'; lightbox.querySelector('img').src = image.src; }); }); </script>

Responsive Design

@media (max-width: 768px) { .gallery { grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 10px; } }

  • Ensure keyboard navigation
  • Provide visible focus indicators
  • Add ARIA labels where needed
  • Ensure sufficient color contrast
  • Support screen readers
