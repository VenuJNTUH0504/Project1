CSS Preprocessors

CSS preprocessors are scripting languages that extend CSS and compile it into regular CSS. They add features like variables, nesting, mixins, and functions, making CSS more maintainable and powerful.

Popular CSS Preprocessors

1. Sass (Syntactically Awesome Style Sheets)

Sass is one of the most popular CSS preprocessors. It uses variables, nesting, mixins, and more to make CSS easier to write and maintain.

// Variables $primary-color: #4a90e2; .button { background: $primary-color; color: #fff; padding: 10px 20px; } /* Compiles to: */ .button { background: #4a90e2; color: #fff; padding: 10px 20px; }

2. LESS

LESS is another preprocessor that extends CSS with dynamic behavior such as variables, mixins, and functions.

// Variables @primary-color: #f39c12; .button { background: @primary-color; color: #fff; padding: 10px 20px; } /* Compiles to: */ .button { background: #f39c12; color: #fff; padding: 10px 20px; }

3. Stylus

Stylus is a flexible preprocessor that allows for both an indented syntax and a CSS-like syntax. It supports variables, mixins, and more.

// Variables primary-color = #23272f .button background: primary-color color: #fff padding: 10px 20px /* Compiles to: */ .button { background: #23272f; color: #fff; padding: 10px 20px; }

Preprocessor Features Comparison

  • Variables: Store values for reuse (colors, sizes, etc.)
  • Nesting: Nest selectors in a way that follows HTML structure
  • Mixins: Reusable chunks of code
  • Functions & Operations: Perform calculations and logic
  • Importing: Split CSS into multiple files
  • Extends/Inheritance: Share styles between selectors
