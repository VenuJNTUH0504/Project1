CSS Grid Advanced Layouts

CSS Grid Layout is a powerful two-dimensional system for web layouts. It allows you to create complex, responsive designs with ease using rows, columns, and grid areas.

Advanced Grid Features

1. Grid Template Areas

Grid template areas let you name sections of your layout and place them visually in the grid.

Header
Sidebar
Content
Footer
display: grid;
grid-template-areas:
  "header header header"
  "sidebar content content"
  "footer footer footer";

2. Minmax and Flexible Tracks

The minmax() function lets you set minimum and maximum sizes for grid tracks, making layouts more flexible.

1fr
minmax(200px, 2fr)
1fr
grid-template-columns: 1fr minmax(200px, 2fr) 1fr;

3. Auto-fit and Auto-fill

These keywords allow grid items to automatically fill the available space, making responsive layouts simple.

Item 1
Item 2
Item 3
Item 4
Item 5
grid-template-columns: repeat(auto-fit, minmax(120px, 1fr));

4. Grid Item Placement

You can place items anywhere in the grid using line numbers or named lines.

.item {
  grid-column: 2 / 4;
  grid-row: 1 / 3;
}

Grid Layouts Comparison

  • Grid Areas: Name and visually arrange sections
  • Minmax: Flexible track sizing
  • Auto-fit/Auto-fill: Responsive, dynamic columns
  • Item Placement: Precise control over item position
  • Nested Grids: Grids inside grid items for complex layouts
