JavaScript Arrays

What are Arrays?

Arrays are ordered collections of values. In JavaScript, arrays can store values of any type, including numbers, strings, objects, and even other arrays.

Note: JavaScript arrays are zero-indexed, which means the first element is at index 0, the second at index 1, and so on.

Creating Arrays

There are several ways to create arrays in JavaScript:

1. Array Literal Notation

// Empty array
const emptyArray = [];

// Array with elements
const fruits = ["apple", "banana", "orange"];

// Array with mixed types
const mixed = [1, "hello", true, { name: "John" }, [1, 2, 3]];

2. Array Constructor

// Empty array
const emptyArray = new Array();

// Array with elements
const numbers = new Array(1, 2, 3, 4, 5);

// Be careful: this creates an array with 3 empty slots
const threeEmptySlots = new Array(3);

3. Array.from() Method

// Create array from string
const chars = Array.from("hello");
console.log(chars); // ["h", "e", "l", "l", "o"]

// Create array from Set
const uniqueNumbers = Array.from(new Set([1, 2, 2, 3, 3, 3]));
console.log(uniqueNumbers); // [1, 2, 3]

// Create array with mapping function
const squares = Array.from([1, 2, 3, 4], x => x * x);
console.log(squares); // [1, 4, 9, 16]

4. Array.of() Method

// Create array with elements
const numbers = Array.of(1, 2, 3, 4, 5);
console.log(numbers); // [1, 2, 3, 4, 5]

// Unlike Array constructor, this creates an array with one element
const singleNumber = Array.of(3);
console.log(singleNumber); // [3]

Accessing Array Elements

You can access array elements using their index:

const fruits = ["apple", "banana", "orange", "grape"];

// Access first element (index 0)
console.log(fruits[0]); // "apple"

// Access third element (index 2)
console.log(fruits[2]); // "orange"

// Access last element
console.log(fruits[fruits.length - 1]); // "grape"

// Using at() method (ES2022)
console.log(fruits.at(0)); // "apple"
console.log(fruits.at(-1)); // "grape" (negative indices count from the end)
Important: If you try to access an element at an index that doesn't exist, JavaScript returns undefined. 
console.log(fruits[10]); // undefined

Modifying Arrays

Arrays in JavaScript are mutable, which means you can change their content after creation.

Changing Elements

const fruits = ["apple", "banana", "orange"];

// Change the second element
fruits[1] = "pear";
console.log(fruits); // ["apple", "pear", "orange"]

Adding Elements

const fruits = ["apple", "banana", "orange"];

// Add element to the end
fruits.push("grape");
console.log(fruits); // ["apple", "banana", "orange", "grape"]

// Add element to the beginning
fruits.unshift("mango");
console.log(fruits); // ["mango", "apple", "banana", "orange", "grape"]

// Add element at a specific position
fruits.splice(2, 0, "kiwi");
console.log(fruits); // ["mango", "apple", "kiwi", "banana", "orange", "grape"]

Removing Elements

const fruits = ["apple", "banana", "orange", "grape", "mango"];

// Remove last element
const lastFruit = fruits.pop();
console.log(lastFruit); // "mango"
console.log(fruits); // ["apple", "banana", "orange", "grape"]

// Remove first element
const firstFruit = fruits.shift();
console.log(firstFruit); // "apple"
console.log(fruits); // ["banana", "orange", "grape"]

// Remove elements at a specific position
const removedFruits = fruits.splice(1, 1);
console.log(removedFruits); // ["orange"]
console.log(fruits); // ["banana", "grape"]

// length property
console.log(fruits.length); // 3

Array Iteration Methods

JavaScript provides several methods for iterating over arrays:

forEach()

const fruits = ["apple", "banana", "orange"];

fruits.forEach((fruit, index) => {
    console.log(`${index}: ${fruit}`);
});
// Outputs:
// 0: apple
// 1: banana
// 2: orange

map()

const numbers = [1, 2, 3, 4, 5];

const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

filter()

const numbers = [1, 2, 3, 4, 5, 6];

const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers); // [2, 4, 6]

reduce()

const numbers = [1, 2, 3, 4, 5];

const sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // 15

find()

const fruits = ["apple", "banana", "orange", "grape"];

const found = fruits.find(fruit => fruit.length > 5);
console.log(found); // "banana"

some()

const numbers = [1, 2, 3, 4, 5];

const hasEven = numbers.some(num => num % 2 === 0);
console.log(hasEven); // true

every()

const numbers = [1, 2, 3, 4, 5];

const allPositive = numbers.every(num => num > 0);
console.log(allPositive); // true

Multidimensional Arrays

JavaScript supports multidimensional arrays, which are arrays of arrays.

// 2D array (matrix)
const matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

// Accessing elements
console.log(matrix[1][2]); // 6 (row 1, column 2)

// Iterating over a 2D array
for (let i = 0; i < matrix.length; i++) {
    for (let j = 0; j < matrix[i].length; j++) {
        console.log(`matrix[${i}][${j}] = ${matrix[i][j]}`);
    }
}

Array Destructuring

Array destructuring allows you to extract values from arrays and assign them to variables.

const colors = ["red", "green", "blue"];

// Basic destructuring
const [first, second, third] = colors;
console.log(first); // "red"
console.log(second); // "green"
console.log(third); // "blue"

// Skip elements
const [primary, , tertiary] = colors;
console.log(primary); // "red"
console.log(tertiary); // "blue"

// Rest pattern
const [head, ...rest] = colors;
console.log(head); // "red"
console.log(rest); // ["green", "blue"]

// Default values
const [a, b, c, d = "yellow"] = colors;
console.log(d); // "yellow"

// Swapping variables
let x = 1;
let y = 2;
[x, y] = [y, x];
console.log(x); // 2
console.log(y); // 1

Spread Operator with Arrays

The spread operator (...) allows you to expand an array into individual elements.

const fruits = ["apple", "banana", "orange"];
const vegetables = ["carrot", "broccoli"];

// Combine arrays
const food = [...fruits, ...vegetables];
console.log(food); // ["apple", "banana", "orange", "carrot", "broccoli"]

// Copy an array
const fruitsCopy = [...fruits];
console.log(fruitsCopy); // ["apple", "banana", "orange"]

// Add elements to a new array
const moreFruits = ["grape", ...fruits, "mango"];
console.log(moreFruits); // ["grape", "apple", "banana", "orange", "mango"]

// Spread in function calls
const numbers = [1, 2, 3];
console.log(Math.max(...numbers)); // 3
