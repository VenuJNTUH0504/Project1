JavaScript Variables

What are Variables?

Variables are containers for storing data values. In JavaScript, variables are used to store data that can be manipulated and referenced throughout your code.

Note: JavaScript is a dynamically typed language, which means you don't need to specify the data type of a variable when you declare it.

Variable Declaration

In JavaScript, there are three ways to declare variables:

1. Using var (older method)

var name = "John";
var age = 30;
var isStudent = true;

2. Using let (introduced in ES6)

let name = "John";
let age = 30;
let isStudent = true;

3. Using const (for constants, introduced in ES6)

const PI = 3.14159;
const MAX_SIZE = 100;
const USER_ROLE = "admin";
Best Practice: In modern JavaScript, it's recommended to use const by default, and only use let when you know the variable's value will change. Avoid using var in new code.

Variable Scope

The scope of a variable determines where in your code the variable is accessible.

Global Scope

Variables declared outside of any function or block have global scope and can be accessed from anywhere in the code.

let globalVar = "I am global";

function testFunction() {
    console.log(globalVar); // Accessible here
}

console.log(globalVar); // Accessible here too

Function Scope

Variables declared inside a function are only accessible within that function.

function testFunction() {
    let functionVar = "I am function-scoped";
    console.log(functionVar); // Accessible here
}

// console.log(functionVar); // Error: functionVar is not defined

Block Scope

Variables declared with let and const inside a block (enclosed by curly braces) are only accessible within that block.

if (true) {
    let blockVar = "I am block-scoped";
    const BLOCK_CONSTANT = 123;
    console.log(blockVar); // Accessible here
    console.log(BLOCK_CONSTANT); // Accessible here
}

// console.log(blockVar); // Error: blockVar is not defined
// console.log(BLOCK_CONSTANT); // Error: BLOCK_CONSTANT is not defined
Important: Variables declared with var are not block-scoped. They are either function-scoped or global.

Variable Hoisting

Hoisting is JavaScript's default behavior of moving declarations to the top of the current scope.

Hoisting with var

console.log(x); // Outputs: undefined (not an error)
var x = 5;
console.log(x); // Outputs: 5

The above code behaves as if it was written like this:

var x; // Declaration is hoisted
console.log(x); // Outputs: undefined
x = 5; // Assignment stays in place
console.log(x); // Outputs: 5

Hoisting with let and const

// console.log(y); // Error: Cannot access 'y' before initialization
let y = 10;
console.log(y); // Outputs: 10

Variables declared with let and const are hoisted but not initialized. The period between the start of the scope and the declaration is called the "temporal dead zone".

Data Types in JavaScript

JavaScript variables can hold different types of data:

Data Type Description Example
String Textual data let name = "John";
Number Numeric data (integers and floating-point) let age = 30; let price = 19.99;
Boolean Logical data (true/false) let isActive = true;
Undefined Variable declared but not assigned a value let result;
Null Intentional absence of any object value let user = null;
Object Complex data structure let person = {name: "John", age: 30};
Array Ordered list of values let colors = ["red", "green", "blue"];
Symbol Unique and immutable primitive value let id = Symbol("id");
BigInt Integers of arbitrary length let bigNumber = 1234567890123456789012345n;

Variable Naming Rules

  • Variable names must start with a letter, underscore (_), or dollar sign ($)
  • Variable names cannot start with a number
  • Variable names can only contain letters, numbers, underscores, or dollar signs
  • Variable names are case-sensitive (name and Name are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as variable names
// Valid variable names
let name = "John";
let _count = 5;
let $price = 19.99;
let firstName = "John";
let last_name = "Doe";

// Invalid variable names
// let 1name = "John"; // Cannot start with a number
// let my-var = 10; // Hyphens are not allowed
// let function = "test"; // Reserved word
