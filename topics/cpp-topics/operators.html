C++ Control Structures

Controlling the flow of program execution

Introduction to Control Structures

Control structures are programming constructs that allow you to control the flow of execution in your program. They enable you to make decisions, repeat actions, and organize code based on different conditions.

C++ provides three main types of control structures:

  1. Selection statements (if, if-else, switch)
  2. Iteration statements (loops: for, while, do-while)
  3. Jump statements (break, continue, goto, return)

Note: Well-designed control structures make your code more readable, maintainable, and efficient. They are fundamental to algorithmic thinking and problem-solving in programming.

Selection Statements

Selection statements allow your program to make decisions based on certain conditions.

if Statement

The if statement executes a block of code if a specified condition is true.

if (condition) {
    // Code to execute if condition is true
}

Example:

int age = 20;

if (age >= 18) {
    std::cout << "You are an adult." << std::endl;
}

if-else Statement

The if-else statement executes one block of code if a condition is true and another block if the condition is false.

if (condition) {
    // Code to execute if condition is true
} else {
    // Code to execute if condition is false
}

Example:

int age = 15;

if (age >= 18) {
    std::cout << "You are an adult." << std::endl;
} else {
    std::cout << "You are a minor." << std::endl;
}

if-else if-else Statement

This structure allows you to check multiple conditions in sequence.

if (condition1) {
    // Code to execute if condition1 is true
} else if (condition2) {
    // Code to execute if condition1 is false and condition2 is true
} else {
    // Code to execute if all conditions are false
}

Example:

int score = 85;

if (score >= 90) {
    std::cout << "Grade: A" << std::endl;
} else if (score >= 80) {
    std::cout << "Grade: B" << std::endl;
} else if (score >= 70) {
    std::cout << "Grade: C" << std::endl;
} else if (score >= 60) {
    std::cout << "Grade: D" << std::endl;
} else {
    std::cout << "Grade: F" << std::endl;
}

Nested if Statements

You can place if statements inside other if statements to create more complex decision-making structures.

if (condition1) {
    // Code to execute if condition1 is true
    
    if (condition2) {
        // Code to execute if both condition1 and condition2 are true
    }
}

Example:

bool hasLicense = true;
int age = 20;

if (age >= 18) {
    std::cout << "You are old enough to drive." << std::endl;
    
    if (hasLicense) {
        std::cout << "You can drive legally." << std::endl;
    } else {
        std::cout << "You need to get a license first." << std::endl;
    }
} else {
    std::cout << "You are too young to drive." << std::endl;
}

Conditional (Ternary) Operator

The conditional operator ?: provides a shorthand way to write simple if-else statements.

condition ? expression1 : expression2;

Example:

int age = 20;
std::string status = (age >= 18) ? "adult" : "minor";
std::cout << "You are a " << status << "." << std::endl;

switch Statement

The switch statement allows you to select one of many code blocks to execute based on the value of a variable or expression.

switch (expression) {
    case value1:
        // Code to execute if expression equals value1
        break;
    case value2:
        // Code to execute if expression equals value2
        break;
    // More cases...
    default:
        // Code to execute if expression doesn't match any case
}

Example:

int day = 3;
switch (day) {
    case 1:
        std::cout << "Monday" << std::endl;
        break;
    case 2:
        std::cout << "Tuesday" << std::endl;
        break;
    case 3:
        std::cout << "Wednesday" << std::endl;
        break;
    case 4:
        std::cout << "Thursday" << std::endl;
        break;
    case 5:
        std::cout << "Friday" << std::endl;
        break;
    case 6:
        std::cout << "Saturday" << std::endl;
        break;
    case 7:
        std::cout << "Sunday" << std::endl;
        break;
    default:
        std::cout << "Invalid day" << std::endl;
}

Warning: Don't forget to include break statements in your switch cases. Without them, execution will "fall through" to the next case, which is rarely what you want.

Iteration Statements (Loops)

Loops allow you to execute a block of code repeatedly as long as a certain condition is true.

for Loop

The for loop is typically used when you know in advance how many times you want to execute a block of code.

for (initialization; condition; update) {
    // Code to repeat
}

Example:

// Print numbers from 1 to 5
for (int i = 1; i <= 5; i++) {
    std::cout << i << " ";
}
// Output: 1 2 3 4 5

while Loop

The while loop executes a block of code as long as a specified condition is true. It checks the condition before executing the loop body.

while (condition) {
    // Code to repeat
}

Example:

// Print numbers from 1 to 5
int i = 1;
while (i <= 5) {
    std::cout << i << " ";
    i++;
}
// Output: 1 2 3 4 5

do-while Loop

The do-while loop is similar to the while loop, but it checks the condition after executing the loop body, ensuring that the code block is executed at least once.

do {
    // Code to repeat
} while (condition);

Example:

// Print numbers from 1 to 5
int i = 1;
do {
    std::cout << i << " ";
    i++;
} while (i <= 5);
// Output: 1 2 3 4 5

Range-based for Loop (C++11)

Introduced in C++11, the range-based for loop provides a simpler way to iterate over elements in a collection (like arrays, vectors, etc.).

for (type variable : collection) {
    // Code to execute for each element
}

Example:

int numbers[] = {1, 2, 3, 4, 5};

// Print each number in the array
for (int num : numbers) {
    std::cout << num << " ";
}
// Output: 1 2 3 4 5

Note: Range-based for loops are more concise and less error-prone than traditional loops when iterating over collections.

Nested Loops

You can place one loop inside another to create nested loops, which are useful for working with multi-dimensional data structures or generating complex patterns.

// Example: Print a 3x3 grid of numbers
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        std::cout << i << "," << j << " ";
    }
    std::cout << std::endl;
}

// Output:
// 1,1 1,2 1,3
// 2,1 2,2 2,3
// 3,1 3,2 3,3

Jump Statements

Jump statements alter the normal flow of program execution by transferring control to another part of the program.

break Statement

The break statement terminates the innermost loop or switch statement and transfers control to the statement immediately following it.

// Example: Find the first even number in an array
int numbers[] = {7, 3, 9, 4, 5, 6};
for (int num : numbers) {
    if (num % 2 == 0) {
        std::cout << "First even number: " << num << std::endl;
        break;  // Exit the loop once an even number is found
    }
}
// Output: First even number: 4

continue Statement

The continue statement skips the rest of the current iteration of a loop and proceeds with the next iteration.

// Example: Print only odd numbers from 1 to 10
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) {
        continue;  // Skip even numbers
    }
    std::cout << i << " ";
}
// Output: 1 3 5 7 9

return Statement

The return statement exits the current function and returns control (and optionally a value) to the calling function.

// Example: Function to check if a number is prime
bool isPrime(int num) {
    if (num <= 1) {
        return false;  // Not prime
    }
    
    for (int i = 2; i * i <= num; i++) {
        if (num % i == 0) {
            return false;  // Not prime
        }
    }
    
    return true;  // Prime
}

// Usage
std::cout << "Is 17 prime? " << (isPrime(17) ? "Yes" : "No") << std::endl;
// Output: Is 17 prime? Yes

goto Statement

The goto statement transfers control to a labeled statement within the same function.

// Example: Using goto (generally not recommended)
int i = 1;
start:
    if (i <= 5) {
        std::cout << i << " ";
        i++;
        goto start;
    }
// Output: 1 2 3 4 5

Warning: The goto statement is generally discouraged in modern programming as it can make code harder to understand and maintain. It's better to use other control structures like loops and if statements.

Comparison of Loop Types

Loop Type When to Use Advantages Disadvantages
for When you know the number of iterations in advance
  • Compact syntax
  • Initialization, condition, and update in one line
  • Good for iterating over sequences
  • Less flexible for complex conditions
  • Can be harder to read with complex expressions
while When you don't know the number of iterations in advance
  • Simple syntax
  • Good for condition-based loops
  • Flexible for various scenarios
  • Initialization and update must be handled separately
  • Risk of infinite loops if condition never becomes false
do-while When you need to execute the loop at least once
  • Guarantees at least one execution
  • Good for input validation
  • Less commonly used
  • Can be confusing for beginners
  • Same risks as while loops
range-based for When iterating over collections (arrays, vectors, etc.)
  • Very concise syntax
  • Eliminates common errors like off-by-one
  • More readable for collection traversal
  • Limited to collections
  • No direct access to index
  • Only available in C++11 and later

Best Practices for Control Structures

General Guidelines

  • Keep it simple: Use the simplest control structure that solves your problem.
  • Avoid deep nesting: Deeply nested control structures are hard to read and maintain. Consider refactoring into separate functions.
  • Use meaningful names: Choose descriptive names for loop variables and conditions.
  • Be consistent: Use a consistent style for braces and indentation.

Selection Statement Tips

  • Use switch instead of multiple if-else statements when comparing a single variable against multiple values.
  • Consider the ternary operator for simple conditional assignments.
  • Always use braces {} for if statements, even for single-line blocks, to prevent bugs when adding more lines later.

Loop Tips

  • Choose the appropriate loop type for your specific needs.
  • Use break and continue judiciously to make your loops more efficient.
  • Be careful with loop conditions to avoid infinite loops.
  • Consider using range-based for loops when working with collections.

Error Prevention

  • Double-check your loop boundaries to prevent off-by-one errors.
  • Ensure that loop variables are properly initialized.
  • Make sure that loop conditions will eventually become false.
  • Be cautious with floating-point comparisons in loop conditions due to precision issues.

Remember: The goal of control structures is to make your code's logic clear and maintainable. Choose the structure that best communicates your intent to other programmers (including your future self).

Common Patterns and Examples

Input Validation

int number;
bool validInput = false;

while (!validInput) {
    std::cout << "Enter a positive number: ";
    std::cin >> number;
    
    if (std::cin.fail() || number <= 0) {
        std::cout << "Invalid input. Please try again." << std::endl;
        std::cin.clear();
        std::cin.ignore(10000, '\n');
    } else {
        validInput = true;
    }
}

std::cout << "You entered: " << number << std::endl;

Finding Maximum/Minimum

int numbers[] = {7, 3, 9, 2, 5, 8};
int size = sizeof(numbers) / sizeof(numbers[0]);

int max = numbers[0];  // Assume first element is maximum
for (int i = 1; i < size; i++) {
    if (numbers[i] > max) {
        max = numbers[i];
    }
}

std::cout << "Maximum value: " << max << std::endl;  // Output: Maximum value: 9

Counting and Summing

int numbers[] = {7, 3, 9, 2, 5, 8};
int size = sizeof(numbers) / sizeof(numbers[0]);

int sum = 0;
int evenCount = 0;

for (int num : numbers) {
    sum += num;
    
    if (num % 2 == 0) {
        evenCount++;
    }
}

double average = static_cast(sum) / size;

std::cout << "Sum: " << sum << std::endl;
std::cout << "Average: " << average << std::endl;
std::cout << "Number of even values: " << evenCount << std::endl;

Nested Loop Pattern

// Print a triangle pattern
int rows = 5;

for (int i = 1; i <= rows; i++) {
    // Print spaces
    for (int j = 1; j <= rows - i; j++) {
        std::cout << " ";
    }
    
    // Print stars
    for (int k = 1; k <= 2 * i - 1; k++) {
        std::cout << "*";
    }
    
    std::cout << std::endl;
}

// Output:
//     *
//    ***
//   *****
//  *******
// *********
