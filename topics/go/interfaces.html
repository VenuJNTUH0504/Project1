Understanding interfaces in Go

Interface Basics

Interfaces in Go define behavior:

// Basic interface
type Speaker interface {
    Speak() string
}

// Implementing interface
type Dog struct {
    Name string
}

func (d Dog) Speak() string {
    return "Woof!"
}

// Using interface
var s Speaker = Dog{Name: "Rex"}
fmt.Println(s.Speak()) // Woof!

Empty Interface

The empty interface can hold any value:

// Empty interface
var i interface{}

// Can hold any type
i = 42
i = "hello"
i = true

// Type assertion
if s, ok := i.(string); ok {
    fmt.Println(s)
}

// Type switch
switch v := i.(type) {
case int:
    fmt.Println("Integer:", v)
case string:
    fmt.Println("String:", v)
default:
    fmt.Println("Unknown type")
}

Interface Composition

Interfaces can be composed of other interfaces:

// Base interfaces
type Reader interface {
    Read(p []byte) (n int, err error)
}

type Writer interface {
    Write(p []byte) (n int, err error)
}

// Composed interface
type ReadWriter interface {
    Reader
    Writer
}

// Implementing composed interface
type File struct {
    // ... fields
}

func (f *File) Read(p []byte) (n int, err error) {
    // ... implementation
}

func (f *File) Write(p []byte) (n int, err error) {
    // ... implementation
}

Interface Values

Understanding interface values and type assertions:

// Interface value
var w io.Writer
w = os.Stdout
w = new(bytes.Buffer)

// Type assertion
var r io.Reader
r = os.Stdin
if w, ok := r.(io.Writer); ok {
    w.Write([]byte("hello"))
}

// Interface satisfaction
type Stringer interface {
    String() string
}

type MyString string

func (s MyString) String() string {
    return string(s)
}

Interface Implementation

Implementing interfaces implicitly:

// Interface definition
type Shape interface {
    Area() float64
    Perimeter() float64
}

// Implementing types
type Circle struct {
    Radius float64
}

func (c Circle) Area() float64 {
    return math.Pi * c.Radius * c.Radius
}

func (c Circle) Perimeter() float64 {
    return 2 * math.Pi * c.Radius
}

type Rectangle struct {
    Width  float64
    Height float64
}

func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}

func (r Rectangle) Perimeter() float64 {
    return 2 * (r.Width + r.Height)
}

Best Practices

  • Keep interfaces small and focused
  • Use interfaces for abstraction
  • Prefer composition over inheritance
  • Document interface requirements
  • Use type assertions carefully
  • Consider interface satisfaction
  • Use empty interfaces sparingly
  • Test interface implementations
