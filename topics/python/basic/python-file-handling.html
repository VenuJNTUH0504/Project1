Python File Handling

Working with files in Python

File Modes

Common file opening modes:

  • 'r' - Read (default)
  • 'w' - Write (truncates file)
  • 'a' - Append
  • 'x' - Create (fails if exists)
  • 'b' - Binary mode
  • 't' - Text mode (default)
  • '+' - Update (read/write)

Basic File Operations

Reading and writing files:

# Writing to a file
with open('example.txt', 'w') as file:
    file.write('Hello, World!\n')
    file.write('This is a new line.')

# Reading a file
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Reading line by line
with open('example.txt', 'r') as file:
    for line in file:
        print(line.strip())

# Appending to a file
with open('example.txt', 'a') as file:
    file.write('\nThis is appended text.')

File Object Methods

Common file object methods:

# Using file object methods
with open('example.txt', 'r') as file:
    # Read first 5 characters
    first_chars = file.read(5)
    print(first_chars)

    # Read next line
    next_line = file.readline()
    print(next_line)

    # Read all remaining lines
    remaining_lines = file.readlines()
    print(remaining_lines)

    # Get current position
    position = file.tell()
    print(f"Current position: {position}")

    # Move to position
    file.seek(0)  # Back to start

Binary Files

Working with binary files:

# Writing binary data
with open('binary.bin', 'wb') as file:
    data = bytes([0, 1, 2, 3, 4, 5])
    file.write(data)

# Reading binary data
with open('binary.bin', 'rb') as file:
    data = file.read()
    print(data)  # Output: b'\x00\x01\x02\x03\x04\x05'

File System Operations

Working with directories and paths:

import os
import shutil

# Directory operations
os.mkdir('new_directory')
os.rmdir('empty_directory')
shutil.rmtree('directory_with_files')

# File operations
os.rename('old.txt', 'new.txt')
os.remove('file_to_delete.txt')

# Path operations
current_dir = os.getcwd()
file_path = os.path.join(current_dir, 'example.txt')
is_file = os.path.isfile(file_path)
is_dir = os.path.isdir(current_dir)

# List directory contents
files = os.listdir(current_dir)
for file in files:
    print(file)

Working with CSV Files

Reading and writing CSV files:

import csv

# Writing CSV
with open('data.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['Name', 'Age', 'City'])
    writer.writerow(['John', '30', 'New York'])
    writer.writerow(['Alice', '25', 'London'])

# Reading CSV
with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

# Using DictReader
with open('data.csv', 'r') as file:
    reader = csv.DictReader(file)
    for row in reader:
        print(row['Name'], row['Age'])

Working with JSON Files

Reading and writing JSON files:

import json

# Writing JSON
data = {
    'name': 'John',
    'age': 30,
    'city': 'New York'
}

with open('data.json', 'w') as file:
    json.dump(data, file, indent=4)

# Reading JSON
with open('data.json', 'r') as file:
    data = json.load(file)
    print(data['name'])  # Output: John

Best Practices

  • Always use context managers (with statement)
  • Handle file exceptions appropriately
  • Close files properly after use
  • Use appropriate file modes
  • Consider file encoding when working with text
  • Use path manipulation functions for cross-platform compatibility
  • Back up important files before operations
  • Use appropriate file formats for different data types
