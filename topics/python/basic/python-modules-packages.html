Python Modules and Packages

Organizing and reusing code in Python

Modules

Creating and using modules:

# mymodule.py
def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

# main.py
import mymodule

print(mymodule.greet("Alice"))  # Hello, Alice!
print(mymodule.add(5, 3))      # 8

# Import specific functions
from mymodule import greet, add

print(greet("Bob"))  # Hello, Bob!
print(add(2, 3))    # 5

# Import with alias
import mymodule as mm

print(mm.greet("Charlie"))  # Hello, Charlie!

Packages

Creating and using packages:

# Directory structure
mypackage/
    __init__.py
    module1.py
    module2.py
    subpackage/
        __init__.py
        module3.py

# __init__.py
from .module1 import function1
from .module2 import function2
from .subpackage.module3 import function3

# Using the package
import mypackage

mypackage.function1()
mypackage.function2()
mypackage.function3()

# Import specific modules
from mypackage import module1
from mypackage.subpackage import module3

Standard Library Modules

Common standard library modules:

# os - Operating system interface
import os
print(os.getcwd())  # Current working directory
os.mkdir('new_dir')

# sys - System-specific parameters
import sys
print(sys.version)  # Python version
print(sys.argv)     # Command line arguments

# math - Mathematical functions
import math
print(math.sqrt(16))  # 4.0
print(math.pi)        # 3.141592653589793

# datetime - Date and time
from datetime import datetime
now = datetime.now()
print(now.strftime("%Y-%m-%d %H:%M:%S"))

# random - Random number generation
import random
print(random.randint(1, 10))  # Random integer
print(random.choice(['a', 'b', 'c']))  # Random choice

# json - JSON data handling
import json
data = {'name': 'John', 'age': 30}
json_str = json.dumps(data)
parsed = json.loads(json_str)

Third-Party Packages

Installing and using third-party packages:

# Install packages using pip
# pip install requests numpy pandas

# requests - HTTP library
import requests
response = requests.get('https://api.example.com/data')
print(response.json())

# numpy - Numerical computing
import numpy as np
array = np.array([1, 2, 3, 4, 5])
print(array.mean())  # 3.0

# pandas - Data analysis
import pandas as pd
df = pd.DataFrame({'A': [1, 2, 3], 'B': [4, 5, 6]})
print(df.describe())

Virtual Environments

Creating and managing virtual environments:

# Create virtual environment
# python -m venv myenv

# Activate virtual environment
# Windows: myenv\Scripts\activate
# Unix/MacOS: source myenv/bin/activate

# Install packages in virtual environment
# pip install package_name

# Deactivate virtual environment
# deactivate

# Requirements file
# pip freeze > requirements.txt
# pip install -r requirements.txt

Relative Imports

Using relative imports in packages:

# Directory structure
mypackage/
    __init__.py
    module1.py
    module2.py
    subpackage/
        __init__.py
        module3.py

# module1.py
from . import module2
from .subpackage import module3

# module3.py
from .. import module1
from ..module2 import function2

Best Practices

  • Use meaningful module and package names
  • Keep modules focused and single-purpose
  • Use __init__.py files appropriately
  • Document modules and packages
  • Use virtual environments for project isolation
  • Maintain requirements.txt for dependencies
  • Follow PEP 8 style guidelines
  • Handle circular imports carefully
  • Use relative imports within packages
  • Test modules independently
