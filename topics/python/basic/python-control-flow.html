Python Control Flow

Understanding control flow statements in Python

If Statements

The if statement is used for conditional execution:

# Basic if statement
age = 18
if age >= 18:
    print("You are an adult")

# if-else statement
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# if-elif-else statement
if age < 13:
    print("You are a child")
elif age < 18:
    print("You are a teenager")
else:
    print("You are an adult")

For Loops

The for loop is used for iterating over a sequence:

# Basic for loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Using range()
for i in range(5):  # 0 to 4
    print(i)

# Nested for loops
for i in range(3):
    for j in range(3):
        print(f"({i}, {j})")

While Loops

The while loop executes as long as a condition is true:

# Basic while loop
count = 0
while count < 5:
    print(count)
    count += 1

# Using break
while True:
    user_input = input("Enter 'quit' to exit: ")
    if user_input == 'quit':
        break
    print(f"You entered: {user_input}")

# Using continue
count = 0
while count < 5:
    count += 1
    if count == 3:
        continue
    print(count)

Break and Continue

The break and continue statements modify the flow of loops:

# Using break in a for loop
for i in range(10):
    if i == 5:
        break
    print(i)

# Using continue in a for loop
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Match-Case Statements

Python 3.10 introduced the match-case statement for pattern matching:

# Match-case example
def handle_command(command):
    match command:
        case "start":
            print("Starting...")
        case "stop":
            print("Stopping...")
        case "pause":
            print("Pausing...")
        case _:
            print("Unknown command")

# Pattern matching with values
def handle_number(number):
    match number:
        case 0:
            print("Zero")
        case 1 | 2 | 3:
            print("Small number")
        case n if n > 100:
            print("Large number")
        case _:
            print("Other number")

Best Practices

  • Use meaningful variable names in conditions
  • Keep conditions simple and readable
  • Avoid deeply nested if statements
  • Use appropriate loop types for different scenarios
  • Include proper error handling
  • Use comments to explain complex logic
  • Consider using early returns to reduce nesting
